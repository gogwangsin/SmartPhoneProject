## 🎮 1. 간단 게임 소개 

### High Concept  
**횡스크롤 플라잉 슈팅 게임 (Horizontal Scrolling Flying Shooter)**  
비행기를 탄 주인공이 오른쪽을 향해 비행하며, 상하좌우로 자유롭게 움직이고, 다가오는 적을 총으로 처치하는 액션 게임입니다.
